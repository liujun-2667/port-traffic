# 港口船舶交通流仿真与安全评估 Dashboard · 产品需求文档 (PRD)

## 1. 产品概述

面向港务调度人员的港口船舶交通流仿真与安全评估系统。通过在不同潮汐与气象条件下模拟航道通行能力、评估船舶交汇风险,辅助调度决策。

- **目标用户**:港务调度员、港口安全管理人员。
- **核心价值**:以可视化的方式量化"到达率/限速/潮位"等参数对吞吐量与安全的影响,在真实调度前预演风险,降低碰撞与拥堵概率。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 港务调度员 | 浏览器访问 | 运行仿真、调整参数、查看动画与报告、回放历史 |
| 系统管理员 | 配置文件 | 维护航道/泊位/锚地/潮汐参数,热更新不重启 |

### 2.2 功能模块

1. **仿真驾驶舱(Dashboard)**:实时 2D 港口俯视动画 + 关键指标面板 + 多类 ECharts 图表。
2. **参数配置**:仿真时长、加速倍率、到达率、潮汐/气象开关,以及航道/港区模型热更新。
3. **参数敏感性实验**:单参数扫描曲线、双参数热力图。
4. **历史回放**:选择历史仿真记录,按时间轴拖动回放轨迹动画。
5. **评估报告**:仿真参数摘要、安全指标统计表、关键事件时间线、瓶颈航段识别、调度建议,支持导出 JSON。

| 页面 | 模块 | 功能描述 |
|------|------|----------|

| 仿真驾驶舱 | 关键指标面板 | 在港船数、等待队列长度、拥堵航段数、累计危险会遇实时数值 |

## 3. 核心流程

调度员进入驾驶舱 → 设置仿真参数(时长/倍率/到达率/潮汐) → 启动仿真 → 引擎按 1 分钟步长推进:生成到达船舶 → 进入航道(检查安全间距/水深) → 航道内推进 → 会遇区域冲突检测 → 抵泊作业 → 作业完成离泊出港 → 实时计算 DCPA/TCPA/拥堵度/等待时间 → 仿真结束生成报告并持久化。

## 4. 用户界面设计

### 4.1 设计风格

- **定位**:工业指挥控制台风(Industrial Command Console)。深色主背景营造夜航调度氛围,辅以高对比数据强调色。
- **主色**:深海军蓝 `#0a1628` 主背景,`#0f2238` 面板;强调色采用航海信号色——青绿 `#00e5c7`(通行/正常)、琥珀 `#ffb547`(拥堵/警示)、警报红 `#ff4d5e`(危险/碰撞)。
- **字体**:标题用 `Oxanium`(工业几何感),数据/正文用 `IBM Plex Sans` + `IBM Plex Mono`(等宽数字便于读数)。
- **布局**:顶栏(标题/倍率/时钟)+ 左侧控制条 + 中央港口画布 + 右侧指标/图表栈,模块卡片化、四角直角带细描边,呼应海图边框。
- **图标**:lucide 图标线性风格;船舶在画布上以几何形状区分类型。

### 4.2 页面设计概览

| 页面 | 模块 | UI 要素 |
|------|------|---------|
| 驾驶舱 | 顶栏 | 标题、仿真时钟、倍率切换(1/5/20/最快)、启动/暂停/重置 |
| 驾驶舱 | 港口画布 | 2D 俯视,航道蓝带,泊位/锚地图标,船舶着色形状,危险连线闪烁 |
| 驾驶舱 | 右侧面板 | KPI 数值卡 + 吞吐折线 + 等待直方 + 安全雷达 + 潮汐曲线 |
| 敏感性 | 扫描配置 | 参数选择/范围/步长,运行,结果曲线/热力图 |
| 回放 | 时间轴 | 刻度条可拖动,播放/暂停,船舶轨迹复现 |
| 报告 | 报告视图 | 摘要+统计表+事件线+瓶颈Top3+建议,导出按钮 |

### 4.3 响应式

桌面优先(调度室大屏),在 1280px 以上最佳;小屏自适应折叠右侧面板。

### 4.4 2D 场景指引

- **视图**:港口水域俯视,坐标系为米,原点左下,画布按比例缩放。
- **航道**:有向蓝色带宽,流向用箭头;拥堵段叠琥珀色透明层。
- **船舶**:集装箱(绿三角)/散货(蓝矩形)/油轮(红菱形)/其他(灰圆),尺寸按船长比例。
- **动效**:船舶按航速平滑移动;危险会遇两船间红色虚线脉冲闪烁;拥堵段呼吸式高亮。
